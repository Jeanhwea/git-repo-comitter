# Plan: Diff 超出 maxInputTokens 时分批提交 LLM 并合并结果

## Context

当前 `generateCommitMessage` 将整个 diff 一次性发送给 LLM,未检查 `maxInputTokens`。当项目变更量大时,可能超出模型上下文窗口导致截断或报错。需要在 diff 超限时自动拆分为多个批次,分别生成局部提交信息,再合并为一条完整的 Conventional Commits 消息。

## 核心流程

```
diff → estimateTokens → 超限?
  ├─ 否 → 走现有 generateCommitMessage (零改动)
  └─ 是 → parseDiffBlocks → groupIntoBatches → 逐批调 LLM(PARTIAL_PROMPT) → 合并调 LLM(MERGE_PROMPT) → validateWithRetry → 最终消息
```

## 实现步骤

### Step 1: 新建 `src/llm/tokens.ts`

- `estimateTokens(text: string): number` — 启发式估算,1 token ≈ 4 chars(对代码 diff 偏保守)

### Step 2: 新建 `src/llm/split.ts`

- `DiffBlock` 接口: `{ filePath, content, estimatedTokens }`
- `DiffBatch` 接口: `{ blocks, content, estimatedTokens }`
- `parseDiffBlocks(diff: string): DiffBlock[]` — 按 `diff --git a/X b/X` 行分割,保留 `=== Staged Changes ===` 等区段头
- `groupIntoBatches(blocks, maxTokens): DiffBatch[]` — 贪心分组,单 block 超限时单独成批

### Step 3: 修改 `src/llm/client.ts`

- `export const SYSTEM_PROMPT` (当前是 const,加 export)
- 新增 `export function createClient(config: AppConfig): OpenAI` — 提取 OpenAI 客户端创建,供 client.ts 和 batch.ts 复用
- `generateCommitMessage` 内部改用 `createClient`

### Step 4: 新建 `src/llm/batch.ts`

- `PARTIAL_SYSTEM_PROMPT` — 告知 LLM 这是部分 diff,生成局部草稿
- `MERGE_SYSTEM_PROMPT` — 告知 LLM 合并多个草稿为一条完整提交信息
- `BatchResult` 接口: `{ message, batchCount }`
- `generateCommitMessageBatched(diff, config): Promise<BatchResult>` — 主入口
- `generatePartialMessage(diffContent, config)` — 单批调用,不校验格式
- `mergePartialMessages(partialMessages, config)` — 合并调用
- `validateWithRetry(merged, partialMessages, config)` — 校验+重试合并(复用 validateCommitMessage)

token 预算计算:
```
effectiveLimit = maxInputTokens - estimateTokens(systemPrompt) - 200(帧开销) - maxOutputTokens
```

合并调用时如果 partialMessages 总 token 超限,截断旧批次并加 `"[... 前面 N 个批次已省略 ...]"` 提示。

### Step 5: 修改 `src/commands/commit.ts`

- 导入改为 `generateCommitMessageBatched`
- 调用处改为 `const { message, batchCount } = await generateCommitMessageBatched(diff, config)`
- batchCount > 1 时输出 `"(已将 diff 分为 N 批次处理并合并)"`

## 文件变更汇总

| 文件 | 操作 | 说明 |
|---|---|---|
| `src/llm/tokens.ts` | 新建 | token 估算 |
| `src/llm/split.ts` | 新建 | diff 解析与分批 |
| `src/llm/batch.ts` | 新建 | 批次编排与合并 |
| `src/llm/client.ts` | 修改 | 导出 SYSTEM_PROMPT,提取 createClient |
| `src/commands/commit.ts` | 修改 | 调用 generateCommitMessageBatched |
