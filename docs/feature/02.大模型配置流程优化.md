# LLM 大模型配置流程优化

## 概述

对 LLM 大模型配置流程进行优化，新增子命令实现初始化后的交互式配置。

## 配置流程

1. **运行初始化子命令**
   执行 `init` 子命令，进入配置流程。

2. **交互式设置参数**
   系统提示用户依次输入以下内容：
   - 大模型链接（API 地址）
   - 模型名称
   - API Key

3. **配置文件存储**
   配置文件将写入 `~/.grc/config.json`。

## 配置文件

- 路径：`~/.grc/config.json`
- 格式：JSON
- 目录 `.grc` 在首次运行时自动创建

## 配置字段

| 字段                   | 说明         | 默认值                          |
|------------------------|--------------|---------------------------------|
| `endpoint`             | API 端点     | `https://api.openai.com/v1`     |
| `apiKey`               | API 密钥     |                                 |
| `llm.model`            | 模型名称     | `deepseek-v4-flash`             |
| `llm.temperature`      | 生成温度     | `0.7`                           |
| `llm.maxInputTokens`   | 最大输入长度 | `262144`                        |
| `llm.maxOutputTokens`  | 最大输出长度 | `16384`                         |
