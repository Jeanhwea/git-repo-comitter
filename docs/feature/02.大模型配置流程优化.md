# LLM 大模型配置流程优化

## 概述

对 LLM 大模型配置流程进行优化，新增子命令实现初始化后的交互式配置。

## 配置流程

1. **运行初始化子命令**
   执行 `init` 子命令，进入配置流程。

2. **交互式设置参数**
   系统提示用户依次输入以下内容：
   - 大模型链接（API 地址）
   - 模型名称
   - API Key

3. **配置文件存储**
   配置文件将写入用户的家目录下的 `.grc` 文件夹中。

## 配置文件路径

- 目录：`~/.grc/`
- 配置文件名：根据具体实现确定（可命名为 `config.json` 或 `config.yaml` 等）

## 实现建议

- 使用命令行框架（如 Click、Argparse）创建子命令 `init`
- 交互式输入可通过 `input()` 或 Prompt 库（如 inquirer）实现
- 配置文件格式建议使用 JSON 或 YAML，便于解析和扩展
- 确保 `.grc` 目录在首次运行时自动创建
